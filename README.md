## userテーブル
|column|Type|Option|
|------|----|------|
|nickname|string|null: false|
|e-mail|string|null: false,unique: true, index:true|
|password|string|null: false, length:{minimu:6}|
|password_conform|string|null:false|
### Association
- has_many : items,dependent:delete_all
